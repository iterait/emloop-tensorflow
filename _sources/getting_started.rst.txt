Getting Started
###############

Before we dive in, cxflow-tensorflow must be properly installed.


Installation
************

The simplest way of installing cxflow-tensorflow is using ``pip``.
This is the recommended option for the majority of users.

.. code-block:: bash

    pip install cxflow-tensorflow

Alternatively, cxflow can be installed directly from the sources.

.. code-block:: bash

    pip install git+https://github.com/Cognexa/cxflow-tensorflow.git

Developer Installation
**********************

Finally, cxflow might be installed directly for development purposes.

.. code-block:: bash

    git clone git@github.com:Cognexa/cxflow-tensorflow.git
    cd cxflow-tensorflow
    pip install -e .
